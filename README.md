## Linear Regression
Linear regression attempts to model the relationship between:

Independent variables (features) (ğ‘‹) and
Dependent variable (target) (y)

The model assumes a linear relationship:
**Linear Regression Equation**
The general form of a linear regression equation is:

$y = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n + \epsilon$

**Where:**

* **y:** Target variable (the value we are trying to predict)
* **x1, x2, ..., xn:** Input features (independent variables)
* **Î²0, Î²1, ..., Î²n:** Coefficients (weights) that determine the influence of each input feature on the target variable. 
    * **Î²0:** Intercept (the value of y when all input features are zero)
* **Ïµ:** Error term (represents the difference between the predicted value of y and the actual value)

This equation represents a linear relationship between the target variable and the input features. The goal of linear regression is to find the values of the coefficients (Î²0, Î²1, ..., Î²n) that best fit the data and minimize the error term.
